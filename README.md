# 🔥 ITACHI VPS DASHBOARD

A simple VPS dashboard script with ITACHI theme.

## Installation

Run:

```bash
bash <(curl -sSL https://raw.githubusercontent.com/nusaybazman41-ops/Repository-Name-my-script/main/install.sh)
```

## Features

- Create Ubuntu VM
- Configure RAM
- Configure CPU Cores
- Configure Disk Size
- TCP Port Forwarding
- Restart VPS
- Clean VPS Files
- ITACHI Premium GUI

## Menu

1. Create & Boot New Ubuntu VPS Instance
2. Restart Existing VPS Instance
3. Modify TCP Port Forward Rules
4. Remove/Clean VPS Cache Files
5. Exit Dashboard
